Program created for the Weekly task for Week 03 Programming and Scripting
Author: Gerry Donnelly

This program take a users string input and outputs the original string and every second letter of the string in reverse order.

Program Operation
1. The program is run as week3task.py.
2. The program requested the user to input a string, can be any alphanumeric characters. 
3. The program reads in the string, prints out the original input string and then prints out every second character starting with the last character and reading backwards. 
4. The program terminates automatically onlce the reverse output has completed.
