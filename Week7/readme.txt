Program to the count number of 'e' characters in the text of the novel Moby Dick
es.py
Author:Gerry Donnelly


This program imports and reads the text into the program, counts the total number of letters in the text, the total number of 'e' characters and outputs the results.

Program Operation
The program es.py is run. 
1. This program requires the user to run thge program as es.py MobyDick.txt. 
2. The text file must be in the same folder as the program. 
3. The program reads in the entire text. 
4. The program counts the total number of characters in the text and also counts the total number of the the letter "e". 
5. The program outputs the results of both counts. 
6. The program terminates when the counts are output.
