collatz.py
Author: Gerry Donnelly

The Program collatz.py takes a positive integer as input, and outputs the successive values of the following calculation.
- If the number divided by 2 is even, divide it by two, 
- If it is odd, multiply it by three and add one. 
- Append each result to the list and output the list when the final result = 1

Program Operation
The program collatz.py is run. 
1. The program asks the user to input a positive integer. 
2. If a negative integer is input this is indicated to the user by an error message and request to re-input.
3. When a postive integer is input the calculations per above description are carried out until thyere is a final remainder. 
4. The program prints out the original number input of the list of integers from the calculation. 
5. The program terminates when the list is output.
